Instruções de navegação

Você já usou uma interface de usuário baseada em texto (TUI - Text-based User Interface)?

Use as téclas direcionais (↑, ↓, ←, →) para navegar entre os links.

Para acessar o menu, combine a tecla Alt com a letra referente ao menu.

Ex.: Alt + H para selecionar Home.

Use Esc para sair do menu.
